<!DOCTYPE html>

This is the doctype declaration. We use this line of text to help the browser determine what version of HTML we are using. The <!DOCTYPE> declaration must be the very first thing in your HTML document, before the <html> tag.


Warning: If you forgot to use the <!DOCTYPE> declaration, a browser may not render your HTML document properly, this is due that it may enter in something called "Quirks" mode and try to deal with your code as if it was written in the late 90s.
